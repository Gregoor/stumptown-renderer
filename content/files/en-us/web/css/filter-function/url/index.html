The url() CSS function uses an SVG filter to change the appearance on the input image.

Syntax

url(location)

Parameters

location
The {{cssxref("<url>")}} of an {{glossary("XML")}} file that specifies an SVG filter, and may include an anchor to a specific filter element.

Example

url(resources.svg#c1)
