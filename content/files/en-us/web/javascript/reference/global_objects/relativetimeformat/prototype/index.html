The Intl.RelativeTimeFormat.prototype property represents the prototype object for the {{jsxref("RelativeTimeFormat", "Intl.RelativeTimeFormat")}} constructor.

{{js_property_attributes(0, 0, 0)}}

Description

See {{jsxref("RelativeTimeFormat")}} for a description of Intl.RelativeTimeFormat instances.

{{jsxref("RelativeTimeFormat", "Intl.RelativeTimeFormat")}} instances inherit from Intl.RelativeTimeFormat.prototype. Modifications to the prototype object are inherited by all {{jsxref("RelativeTimeFormat", "Intl.RelativeTimeFormat")}} instances.
