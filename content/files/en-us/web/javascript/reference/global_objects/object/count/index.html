The __count__ property used to store the count of enumerable properties on the object, but it has been removed.

Syntax

obj.__count__

Examples

{ 1: 1 }.__count__              // 1
[].__count__                    // 0
[1].__count__                   // 1
[1, /* hole */, 2, 3].__count__ // 3

Specifications

Not part of any specification.
