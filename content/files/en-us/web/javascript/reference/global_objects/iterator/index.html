The Iterator global object is an abstract class providing helper methods, similar to the ones exposed on {{JSxRef("Array")}} instances.

All built-in iterators inherit from the Iterator abstract class.

{{JSxRef("Iterator.Iterator", "Iterator()")}} {{Experimental_Inline}}
