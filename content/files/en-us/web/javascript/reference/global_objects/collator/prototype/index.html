The Intl.Collator.prototype property represents the prototype object for the {{jsxref("Collator", "Intl.Collator")}} constructor.

{{js_property_attributes(0, 0, 0)}}

Description

See {{jsxref("Collator")}} for a description of Intl.Collator instances.

{{jsxref("Collator", "Intl.Collator")}} instances inherit from Intl.Collator.prototype. Modifications to the prototype object are inherited by all {{jsxref("Collator", "Intl.Collator")}} instances.
