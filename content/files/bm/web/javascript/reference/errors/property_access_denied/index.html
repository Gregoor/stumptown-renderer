{{jsSidebar("Errors")}}

Message

Erro: Acesso negado à propriedade "x"

Tipo de Erro

{{jsxref("Error")}}.

Que aconteceu de errado?

There was attempt to access an object for which you have no permission. This is likely an {{HTMLElement("iframe")}} element loaded from a different domain for which you violated the same-origin policy.

Exemplos

<!DOCTYPE html>
<html>
  <head>
    <iframe id="myframe" src="http://www1.w3c-test.org/common/blank.html"></iframe>
    <script>
      onload = function() {
        console.log(frames[0].document);
        // Erro: Acesso negado à propriedade "document"
      }
    </script>
  </head>
  <body></body>
</html>
